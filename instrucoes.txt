Para criar projeto pelo Visual studio Code

0 - Criar projeto no ponto ASP.NET Core Web API, que tem a chavezinha apontada para o lado esquerdo com C# em cima.

1 - Definir o nome e tal. Dar Ok

2 - Definir framework .NET 6.0, sem autentificação, definir com configurar para HTTPS, definir sem docker e definir com os 3 espaços de opções no final. 

3 - no nome do projeto na aba a direita clicar com o botão direito no nome do projeto que é SistemaDeTarefas e escolher a opção Gerenciar Pacote Nugets.

4 - Pesquisar no Browse o Microsoft.EntityFrameworkCore e baixar a versão 6.0.0. Instalar e confirmar a versão.

5 - Instalar o Microsoft.EntityFrameworkCore.Design e baixar a versão 6.0.0. Instalar e confirmar a versão.

6 - Instalar o Microsoft.EntityFrameworkCore.SqlServer e baixar a versão 6.0.0. Instalar e confirmar a versão.

7 - Instalar o Microsoft.EntityFrameworkCore.Tools e baixar a versão 6.0.0. Instalar e confirmar a versão.
